# Fluffy core
Is a TS library for simplifying communication between microservices
in microservice architecture. It supports multiple providers such as
NATS, Kafka, etc. and provides unified interface for fast integration
in a project.

# THIS PROJECT IS STILL UNDER DEVELOPMENT
# PLEASE, DO NOT USE IT IN PRODUCTION!!!!
## Currently, only NATS provider is implemented and tested
Please use it carefully and report any issues you find. I'll do my best to fix them.
But some things may not work as expected at current stage.
